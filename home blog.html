Site SSC

O seu site de musica catóilica

Sobre o projeto:

O projeto surgiu após momentos de conversas e ser percebido que existe uma gama de letras / cifras católicas escritos de forma erradas, tendo isso em vista sentiu-se a necessidade da criação de um local para armazenar músicas (católicas) separadas por categorias, e outra vantagem é que no mesmo site apresenta letras e músicas sem a necessidade de pegar de outra fonte causando desencontros entre os musicos.
Se você chegou aqui via Github e é programador sinta-se a vontade para sugerir alterações via pull-request
